Welcome to Number Guess Game!

The purpose of this game is to guess the number that picked by the computer.

For each guess of yours, the computer gives you a feedback, so that you can modify your guess in the next round.

You should interpret the feedbacks as follows:

  • + x : x number of digits are among digits of the true number and in the correct location.

  • - x : x number of digits are among digits of the true number but not in the correct location.

Choose the number of digits you want to play with!
Press START to initialize the game!
Press GUESS to make a guess on the true number!
